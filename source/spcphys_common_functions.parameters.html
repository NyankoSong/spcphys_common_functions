Calculate the Alfvenic parameters.

Parameters:

  • p_date (List[datetime] or numpy.ndarray) – List or array of datetime objects for proton velocity data

  • v (astropy.units.Quantity) – Proton velocity data in shape (time, 3)

  • n (astropy.units.Quantity) – Proton number density data in shape (time)

  • b_date (List[datetime] or numpy.ndarray) – List or array of datetime objects for magnetic field data

  • b (astropy.units.Quantity) – Magnetic field data in shape (time, 3)

  • least_data_in_window (int or float, optional) – Least number of valid data points, defaults to 20

  • down_sampling_method (str, optional) – Method for down-sampling the magnetic field data to match the proton velocity data, either ‘interpolate’ or ‘mean’, defaults to ‘interpolate’

  • down_sampling_window (datetime.timedelta or None, optional) – Time window for down-sampling the magnetic field data, defaults to None, which means it is determined by the resolution of the proton velocity data

Returns:

Dictionary containing Alfvenic parameters (r3, p3, residual_energy, cross_helicity, alfven_ratio, compressibility, vA, num_valid_p_points, num_valid_b_points)

Return type:

dict

This function calculates the correlation coefficient between velocity and magnetic field, residual energy, cross helicity, Alfven ratio, and compressibility.
